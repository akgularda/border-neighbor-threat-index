BNTI Methodology

1. Data Collection

The system continuously monitors multiple RSS/Atom feeds from regional news outlets, wire services, and specialized publications for each target country. Sources include local newspapers in native languages, international news agencies, and regional security-focused media. Data is fetched at hourly intervals with automatic retry mechanisms and cached to prevent duplicate processing.

2. Natural Language Processing

Each article is processed using XLM-RoBERTa-Large-XNLI, a multilingual transformer model trained for zero-shot classification. This enables threat categorization across 40+ languages without requiring language-specific training data.

Articles are classified into threat categories: military_conflict, terrorism, political_instability, border_security, refugee_crisis, civil_unrest, and economic_crisis, diplomatic_tensions, natural_disaster.

3. Weighted Scoring System

The scoring algorithm is based on a Modified Goldstein Scale, adapted for regional threat assessment. Each threat category carries a predefined weight reflecting its potential security impact:

military_conflict: +8.0
terrorism: +7.0
political_instability: +4.0
border_security: +5.0
diplomatic_tensions: +2.5
trade_agreement: -2.0

5. Threat Classification Thresholds

Stable: 1.0 - 3.9
Elevated: 4.0 - 6.9
Critical: 7.0 - 10.0

6. Forecasting

Short-term forecasting uses Weighted Linear Regression with exponentially decaying weights favoring recent observations. This provides directional trend indicators while accounting for the inherent uncertainty in geopolitical predictions.
